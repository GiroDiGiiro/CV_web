Récolement des Données liées à l'entretien des marais

QGIS Python PyQT PyQGIS PgAdmin SQL WebSig - LizMap HTML/CSS Bash QField Tablettes

Présentation

L'UNIMA souhaite améliorer la communication avec ses membres, en particulier les Associations Syndicales de propriétaires du marais et le conseil départemental, concernant le suivi des travaux d’entretien du réseau hydraulique. La solution technique doit intégrer et historiser les informations des travaux dans une base de données, puis les puis les diffuser via le WebSIG.

Fonctionnement de l'application

L'application développée est destinée à trois acteurs :

  • Le Bureau d'études
  • La régie de travaux
  • Les conducteurs d'engin mécanisé

Et est composée de six éléments techniques :

  • Un plugin QGIS codé en Python
  • Un serveur externe (QFieldCloud)
  • Du matériel portable avec les tablettes terrains
  • Une application Android (QField)
  • Trois bases de données
  • Un WebSig Lizmap

Trois langages de programmation ont été utilisés

  • SQL - Pour extraires les données depuis les BD
  • BASH - Pour automatiser la sauvegarde et le transfert de données entre BD
  • Python - Plugin Qgis

Résultats

Cette application a permis de standardiser la planification des travaux d'entretien du marais, de faciliter l'échange d'informations entre les différents acteurs ainsi que d'automatiser la sauvegarde et la diffusion des informations.

Conclusion

Ce projet de récolement des données sur l'entretien des marais montre comment des solutions géomatiques peuvent être adaptées aux besoins des acteurs locaux.

Le développement du plugin QGIS en Python et l'utilisation de bases de données géospatiales ont permis d’automatiser des processus clés et d'assurer un suivi précis des interventions. La solution WebSIG, associée à des outils mobiles comme QField, a facilité la gestion en temps quasi réel des données, augmentant ainsi flexibilité et transparence.

En résumé, ce projet illustre une solide maîtrise des outils géomatiques et la capacité à intégrer plusieurs technologies pour offrir des solutions pratiques et innovantes pour la gestion des territoires.
