Plugin d'extraction d'informations cadastrales

Python PyQT PyQGIS Cadastre

Présentation

La Communauté de communes de Quimperlé effectue souvent du publipostage pour prévenir ses habitants de différentes informations, notamment lors de la réalisation de travaux sur l'espace public. Le pôle SIG a donc demandé un plugin QGIS afin d'extraire et de formater automatiquement les différentes informations nécessaires au publipostage.

Fonctionnement du Plugin

Le plugin devra se connecter à la base de données cadastrale de la communauté de communes, puis sélectionner la couche à intersecter (une couche de travaux, par exemple). L'utilisateur pourra ensuite sélectionner un chemin de sortie, une projection et le type de fichier souhaité.

Les informations seront regroupées en deux groupes :

  • Les informations parcellaires géoréférencées
  • Les informations des propriétaires formatées pour le publipostage
